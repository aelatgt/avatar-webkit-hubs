# Setup

Configure environment variables before any NVM configuration:

```bash
# .bashrc
export AVATAR_WEBKIT_REPO_SECRET="..."
export AVATAR_WEBKIT_AUTH_TOKEN="..."
```

# Development

```bash
yarn
yarn dev
```
